************
SoC Targets
************
SoC Targets are systems that use the RISC-V atom core alongwith different peripherals. From a 
complexity and functionality point-of-view, SoC Targets can be as simple as a wrapper to the core 
interface (e.g. AtomBones) and as complex as multicore fully-fledged SOCs.
